### Step 1: Create a GitHub Repository

1. Go to [GitHub](https://github.com) and sign in
2. Click the **+** icon in the top right â†’ **New repository**
3. Name your repository (e.g., `audio-player`)
4. Make it **Public**
5. Click **Create repository**

### Step 2: Upload Your Files

Open a terminal in this folder and run:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

Replace `YOUR_USERNAME` and `YOUR_REPO_NAME` with your actual GitHub username and repository name.

### Step 3: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** tab
3. Scroll down to **Pages** section (in the left sidebar under "Code and automation")
4. Under **Source**, select **main** branch
5. Click **Save**
6. Wait a few minutes for the site to deploy

Your website will be live at: `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`
